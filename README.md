# Whatsapp-Web Clone

Project Link - https://whatsapp-mern-857d8.web.app/

## About
- WhatsApp-web clone made using the MERN stack and Pusher API.

## Features 
-  Google authentication
-  Real-time chat 
-  Creating any number of groups
-  Simple end-to-end encryption.
